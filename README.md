# Wally App

Spanish-first reservation and payment platform for a sports center in La Paz,
Bolivia.

Wally App manages two courts for wally, racket, and fútbol 2. Customers can
browse availability, reserve a court, pay, and view their reservations. Admin
users can manage courts, schedules, prices, reservations, and payments.

Preferred stack: Next.js, TypeScript, Tailwind CSS, Supabase PostgreSQL, and
Prisma.
